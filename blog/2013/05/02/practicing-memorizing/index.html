Practicing Memorizing

May 2nd, 2013

I’m working on my ability to remember information. Instead of using memory “tricks” like the method of loci or chunking, I’m working on simply using a “brute force” approach to storing information in long term memory using flash cards.

Although this isn’t the most efficient way to remember information, it’s the best way to remember information unintentionally.

The problem with the method of loci and chunking is simple: those techniques only provide you with access to information within the context of the technique. The method of loci makes memorizing information very easy, but you need to focus on visualizing a spatial location to access your memories. Chunking groups small pieces of information into larger parts, but what if you need an individual piece at a random time?

The problem with a lot of memory techniques is that they don’t address the main issue with memory: when is it useful to know the information? Chunking and the method of loci are useful for memorizing a speech or for situations when you know in advance that you will need a piece of information.

What if I need information, but I can’t predict when or how that need will arise?

The best way, in that situation, is to create as many associations with related things as possible. As a result, I’m giving a few different flash card tools a try. With these tools, I’m attempting to memorize maps of the world, vocabulary words, historical events, and animal species. To make the process as effective as possible, I’m working on creating a set of images from Flickr corresponding to each of the words, and the flash cards will have multiple “definitions” that can point to a single word.

So far, I’ve enjoyed using the following tools:

  • Anki - a downloadable tool that tracks the intervals between correct and incorrect recall to maximize the probability of successfully storing a flash card in long term memory
  • Quizlet - a web-based social tool with thousands of existing sets of flash cards that are ripe for memorizing and learning
  • Memrise - a newer and sleeker web-based tool with fewer flash cards available to study, but incorporates gamification and tracking of accuracy to maximize recall
