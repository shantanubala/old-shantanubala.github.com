Coursera Machine Learning

May 6th, 2013

I’m two weeks into the Stanford Machine Learning course taught by Andrew Ng through Coursera.

The class is excellent, and I’m having a great time going through the material. Coursera’s platform not only makes it really easy to watch the lectures, but the material in the course is taught really well.

Overall, the experience has been overwhelmingly positive.

There are two types of assignments in the class: lecture review questions and programming exercises. The lecture review questions are multiple choice questions you complete online, and the programming exercises are automatically-graded code exercises that focus on Octave.

So far, the material has been relatively simple, and the course has only covered a brief introduction to the topic of Machine Learning as well as linear regression with a single or multiple variables. The Octave programming language has also been introduced, but the course is still progressing quite well.

The high quality of the course and the amazing price tag (free!) seem to be signalling pretty big changes in education coming soon.
